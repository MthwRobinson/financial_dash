# Financial Dash 
:penguin:

Just something I worked on to get a feel for the new plotly frame work for python

:ox:

To clone the repo:

`git clone git@github.com:MthwRobinson/financial_dash.git`

To run tests:

```
cd ~/financial_dash
python -m pytest
```

To run the app

```
cd ~/financial_dash/financial_dash
python app.py
```
